# Dashboard Sales Forecast - Varejo | Toronto

**Setor:** Varejo / E-commerce  
**Objetivo:** Análise de vendas e previsão de demanda para lojas de Ontario

**KPIs Principais:**
- Receita Total e Margem Bruta em CAD$
- Ticket Médio por Loja
- Giro de Estoque / Inventory Turn
- Análise RFM de Clientes
- Forecast de Vendas 3 meses

**Tecnologias Power BI:**
- **DAX:** Time Intelligence, SAMEPERIODLASTYEAR, USERELATIONSHIP
- **Power Query:** Conexão API Shopify + Excel
- **Visuais:** Gráfico de Linha com Forecast, Treemap, Mapa
- **Recursos:** What-If Parameters, Drill-down

**Destaque Canadá:** Padrão usado por grandes redes de Toronto.
**Status:** Em desenvolvimento - Print e .pbix serão adicionados em breve.
