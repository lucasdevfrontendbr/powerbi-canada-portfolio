# Dashboard Patient Flow - Saúde | Toronto

**Setor:** Hospitais / Clínicas  
**Objetivo:** Monitorar fluxo de pacientes e otimizar tempo de espera, padrão PHIPA

**KPIs Principais:**
- Taxa de Ocupação de Leitos %
- Tempo Médio de Espera na Emergência
- Taxa de Readmissão 30 dias
- Custo Médio por Paciente em CAD$
- SLA de Atendimento

**Tecnologias Power BI:**
- **DAX:** Medidas com RLS, CALCULATE com múltiplos filtros
- **Power Query:** Conexão HL7 / SQL Server Hospitalar
- **Visuais:** Funil, KPI Card, Gauge, Heatmap
- **Recursos:** Row-Level Security para compliance PHIPA

**Destaque Canadá:** Conformidade com PHIPA - lei de privacidade de saúde de Ontario.
**Status:** Em desenvolvimento - Print e .pbix serão adicionados em breve.
