# Dashboard Budget Tracking - Construção Civil | Toronto

**Setor:** Construtoras / Obras  
**Objetivo:** Acompanhar Orçado vs Realizado e cronograma de obras em Toronto

**KPIs Principais:**
- % Orçado vs Realizado por Fase
- Desvio de Custo em CAD$
- Cronograma Físico-Financeiro
- Índice de Acidentes de Trabalho
- Curva S do Projeto

**Tecnologias Power BI:**
- **DAX:** Variância, Projeção, SUMX
- **Power Query:** Conexão MS Project + Excel
- **Visuais:** Gráfico Gantt, Waterfall, Medidor, Tabela
- **Recursos:** Bookmarks para Cenários, Drill-through

**Destaque Canadá:** Padrão usado por construtoras de GTA - Greater Toronto Area.
**Status:** Em desenvolvimento - Print e .pbix serão adicionados em breve.
